# AI Document Chatbot (RAG-based)

This project is a Retrieval-Augmented Generation (RAG) chatbot that allows users to upload documents and ask questions based on their content.

## Features
- Upload PDF documents
- Ask questions from documents
- Context-aware answers using LLMs

## Tech Stack
- Python
- LangChain
- FAISS (Vector Database)
- OpenAI / Gemini API
- Streamlit (for UI)

## Status
🚧 Work in Progress – actively building

## Goal
To build an AI-powered document assistant using modern LLM techniques.
